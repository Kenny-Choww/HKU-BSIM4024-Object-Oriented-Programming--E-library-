# HKU-BSIM4024-Object-Oriented-Programming-E-library
This is a course work of The University of Hong Kong (HKU)

# E-Library System

A Python-based E-Library System developed for HKU BSIM4024 using object-oriented programming.  
The project supports item management, user management, borrowing, returning, and enquiry functions for books, periodicals, newspapers, and multimedia resources.

**Live Demo:** [Streamlit App](http://hku-bsim4024-e-library-group1.streamlit.app/)

## Main OOP Concepts
- Inheritance
- Encapsulation
- Polymorphism
- Object relationships

## Tech Stack
- Python
- Streamlit
- MongoDB
